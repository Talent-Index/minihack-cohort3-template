# Mini Hack Cohort 3 - Agentic AI Systems on Avalanche

Fork this repo. Build an AI agent that connects to Avalanche. Deploy it.

## Prerequisites

Cohort 1 completion or equivalent: ethers.js, smart contract deployment, Node.js async programming.

## Getting started

```bash
git clone https://github.com/YOUR-HANDLE/minihack-cohort3-template.git
cd minihack-cohort3-template
npm install
cp .env.example .env
npx hardhat compile
```

## Weekly deliverables

| Week | Deliverable |
|------|-------------|
| 1 | CLI agent using Claude API with at least one external tool |
| 2 | Wallet advisor agent - reads Fuji transaction history, generates AI summary |
| 3 | RAG agent from 5+ docs + autonomous USDC payment trigger on Fuji |
| 4 | Final deployed agentic system - public URL + architecture diagram + demo video |
